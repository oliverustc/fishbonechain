# Pianist 协议参考手册

> 来源：pianist.pdf（Pianist: Scalable zkRollups via Fully Distributed Zero-Knowledge Proofs）

---

## 一、总体架构

**目标**：M 台机器分布式生成 Plonk 证明，通信量 O(1) per machine，证明大小 O(1)，验证时间 O(1)。

**系统角色**：
- P₀：主节点（master），负责参数分发、聚合、交互验证者
- P₁...P_{M-1}：子节点（worker），各持有子电路 Cᵢ 的见证

**电路划分**：总规模 N，分为 M 个子电路，每个规模 T = N/M。

## 二、二元多项式约束系统

### 2.1 从 Plonk 到二元多项式

每个子节点 Pᵢ 持有单变量多项式 aᵢ(X), bᵢ(X), oᵢ(X)（T 次，定义在 T 次单位根 ΩX 上）。

主节点将 M 个子电路的多项式**通过拉格朗日基聚合**为二元多项式：
```
A(Y,X) = Σᵢ Rᵢ(Y)·aᵢ(X)
```
其中 Rᵢ(Y) 为 M 次单位根 ΩY 上的拉格朗日基多项式（close-form：Rᵢ(Y) = ωY^i/M · (Y^M - 1)/(Y - ωY^i)）。

**关键性质**：A(ωY^i, X) = aᵢ(X)，即 Y 轴第 i 个单位根处退化为子电路的单变量多项式。

### 2.2 门约束

每个子电路满足 Plonk 门约束：
```
gᵢ(X) := qa,i(X)·aᵢ(X) + qb,i(X)·bᵢ(X) + qo,i(X)·oᵢ(X)
         + qab,i(X)·aᵢ(X)·bᵢ(X) + qc,i(X) = 0  (∀X ∈ ΩX)
```

二元聚合门约束：
```
G(Y,X) := Qa(Y,X)A(Y,X) + Qb(Y,X)B(Y,X) + Qab(Y,X)A(Y,X)B(Y,X)
         + Qo(Y,X)O(Y,X) + Qc(Y,X)
```

### 2.3 复制约束（置换论证）

引入挑战 η, γ，定义：
```
fᵢ(X) := (aᵢ(X) + η·σa,i(X) + γ)(bᵢ(X) + η·σb,i(X) + γ)(oᵢ(X) + η·σo,i(X) + γ)
fᵢ'(X) := (aᵢ(X) + η·ka·X + γ)(bᵢ(X) + η·kb·X + γ)(oᵢ(X) + η·ko·X + γ)
```

运行乘积多项式（子节点本地计算）：
```
zᵢ(ωX^j) := ∏_{k=0}^{j-1} fᵢ(ωX^k) / fᵢ'(ωX^k)
```

置换约束多项式：
```
P₀(Y,X) := L₀(X)·(Z(Y,X) - 1)
P₁(Y,X) := Z(Y,X)·F(Y,X) - Z(Y,ωX·X)·F'(Y,X)
```

### 2.4 统一约束等式

引入随机数 λ，所有约束聚合为：
```
G(Y,X) + λ·P₀(Y,X) + λ²·P₁(Y,X) - VX(X)·HX(Y,X) = VY(Y)·HY(Y,X)
```
其中 VX(X) = X^T - 1，VY(Y) = Y^M - 1。

子电路商多项式 hᵢ(X)（X 次数 ≤ 3T-2）：
```
hᵢ(X) = (gᵢ(X) + λ·p_{i,0}(X) + λ²·p_{i,1}(X)) / (X^T - 1)
HX(Y,X) = Σᵢ Rᵢ(Y)·hᵢ(X)  （分3个承诺块 HX,0, HX,1, HX,2）
```

## 三、DKZG 承诺方案

### 3.1 KeyGen

生成公共参数（SRS）：
```
g, g^{τX}, g^{τY}
U = (Uᵢ)_{0≤i<M} where Uᵢ,ⱼ = g^{Rᵢ(τY)·Lⱼ(τX)}
V = (g^{Rᵢ(τY)})_{0≤i<M}
```

### 3.2 Commit（分布式）

子节点 Pᵢ 计算本地承诺：
```
comf_i = ∏_{j=0}^{T-1} Uᵢ,ⱼ^{fᵢ,ⱼ}
```
主节点聚合：`comF = ∏ᵢ comf_i`

等价于：comF = g^{F(τY, τX)}（全局双变量多项式在陷门处的承诺）

### 3.3 Open(F, β, α)

1. **X轴开启**（每个子节点并行）：
   - 计算 fᵢ(α)，商多项式 q₀^(i)(X) = (fᵢ(X) - fᵢ(α))/(X-α)
   - 发送 fᵢ(α), π₀^(i) = g^{Rᵢ(τY)·q₀^(i)(τX)} 给主节点

2. **主节点聚合 X 轴**：
   - π₀ = ∏ᵢ π₀^(i)（= g^{Q₀(τY,τX)}）
   - 恢复 F(Y,α) = Σᵢ Rᵢ(Y)·fᵢ(α)

3. **Y轴开启**（主节点）：
   - q₁(Y) = (F(Y,α) - F(β,α))/(Y-β)
   - π₁ = g^{q₁(τY)}
   - z = F(β,α)

### 3.4 Verify

```
e(comF / g^z, g) = e(π₀, g^{τX-α}) · e(π₁, g^{τY-β})
```
（2 个配对检查）

## 四、Pianist 协议流程

### 4.1 初始化

- KeyGen：生成 SRS（U, V, g^{τX}, g^{τY}）
- 主节点预计算选择器多项式的 DKZG 承诺 comS_pp
- 验证密钥：vk = (g^{τX}, g^{τY}, comS_pp)

### 4.2 承诺阶段

| 步骤 | 操作 |
|------|------|
| 子节点 | 计算 comA_i, comB_i, comO_i，发给主节点 |
| 主节点 | 聚合 comA = ∏ comA_i，派生 η, γ（FS变换） |
| 子节点 | 计算 comZ_i，发给主节点 |
| 主节点 | 聚合 comZ，派生 λ |
| 子节点 | 计算 comH_i（商多项式），发给主节点 |
| 主节点 | 聚合 comHX（3块），派生 α |

### 4.3 证明阶段（开启 X 轴）

子节点计算：
- aᵢ(α), bᵢ(α), oᵢ(α), zᵢ(α), hᵢ(α)（在 α 处求值）
- zᵢ(ωXα)（在 ωXα 处求值）
- X 轴开启证明 π₀^(i)（对每个多项式）

主节点聚合 → 得到 F(Y,α)（每个多项式的 Y 轴切片），并获得 HY(Y,X)，派生 β。

### 4.4 证明阶段（开启 Y 轴）

主节点对每个 F(Y,α) 执行 Y 轴开启：
- F(β,α)：最终标量值
- π₁,S：Y 轴开启证明

### 4.5 最终证明格式

```
π = {
  承诺: comA, comB, comO, comZ, comHX(3块), comHY(3块)
  X轴证明: π₀,A, π₀,B, π₀,O, π₀,Z, π₀,HX, π₀,Z'  (6个G₁)
  Y轴证明: π₁,A, π₁,B, π₁,O, π₁,Z, π₁,HX, π₁,HY, π₁,Z'  (7个G₁)
  求值: A(β,α), B(β,α), O(β,α), Z(β,α), HX(β,α), HY(β,α), Z(β,ωXα)
}
```

### 4.6 验证

1. **FS 重算**：重算 η, γ, λ, α, β
2. **承诺验证**（对每个多项式 S 执行 DKZG.Verify）：
   ```
   e(comS / g^{S(β,α)}, g) = e(π₀,S, [τX-α]₂) · e(π₁,S, [τY-β]₂)
   ```
   偏移点 ωXα 处额外验证 Z(β,ωXα)。
3. **代数约束检查**：
   ```
   G(β,α) + λ·P₀(β,α) + λ²·P₁(β,α) - VX(α)·HX(β,α) = VY(β)·HY(β,α)
   ```

## 五、复杂度

| 指标 | Pianist |
|------|---------|
| 子节点证明时间 | O(T log T)𝔽 + O(T)𝔾 |
| 主节点额外开销 | O(M log M)𝔽 + O(M)𝔾 |
| 证明大小 | O(1)|G|（约2.2KB） |
| 验证时间 | O(1)𝕡 + O(log N)𝔽（DKZG验证） |
| 子节点通信量 | O(1)|G|（约2.1KB per node） |

## 六、BPiano 相比 Pianist 的改动

| 组件 | Pianist | BPiano |
|------|---------|--------|
| X轴开启 | 分别开启 α 和 ωXα（2套证明） | Shplonk 聚合为 comQX（1个） |
| Y轴开启 | 分别开启每个 Y 多项式（7个） | 聚合为 GY(Y)，1个 π₁,agg |
| 验证配对 | ~14次（每多项式2次） | 4次（合并） |
| 证明大小 | ~2.2KB | ~0.6KB |
| 批量K个证明 | K × 4次配对 | 4次配对（与K无关） |

## 七、符号速查

| 符号 | 含义 |
|------|------|
| N | 总电路规模 |
| M | 子节点数量 |
| T = N/M | 单个子电路规模 |
| ΩX | T 次单位根集合 |
| ΩY | M 次单位根集合 |
| Lⱼ(X) | X 轴第 j 个拉格朗日基（T 次单位根） |
| Rᵢ(Y) | Y 轴第 i 个拉格朗日基（M 次单位根） |
| τX, τY | KZG 陷门（setup 后销毁） |
| α | X 轴挑战点（由 comHX 派生） |
| β | Y 轴挑战点（由 α 后续承诺派生） |
| ωX | T 次单位根的生成元 |
| ωY | M 次单位根的生成元 |
| η, γ | 置换论证挑战 |
| λ | 约束聚合挑战 |
| ν | BPiano Shplonk 聚合挑战 |
| μ | BPiano Y 轴聚合挑战 |
| ρ | BPiano 配对合并随机数 |
